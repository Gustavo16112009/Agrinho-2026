Agrinho 2026: [Quiz do Agro]

1 - Descrição do projeto

Este projeto foi desenvolvido para o Concurso Agrinho 2026, na categoria Programação, subcategoria Ensino Médio. O site aborda o tema “Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente”, apresentando um jogo interativo no formato de Quiz com 20 perguntas dinâmicas focadas em práticas reais de preservação do solo, economia de água, uso de tecnologias como drones e controle biológico de pragas.

2 - Objetivo

O objetivo deste projeto é conscientizar e testar o conhecimento do usuário sobre como a tecnologia e as boas práticas de manejo no campo permitem que o agronegócio continue produzindo alimentos em larga escala enquanto protege e recupera o meio ambiente. ### Público-alvo.

Público-alvo
O projeto é destinado a estudantes, professores e à comunidade em geral interessados em compreender a importância do equilíbrio entre a produção agropecuária e a preservação ambiental de uma forma leve, interativa e gamificada.

3 - Estrutura do projeto

O projeto é composto pelos seguintes arquivos e pastas:

 - `index.html`: arquivo principal com a estrutura física das telas do jogo (Início, Quiz e Fim); - `style.css`: arquivo responsável pelo design visual moderno, paleta de cores voltada à natureza e responsividade;
- `script.js`: arquivo responsável por toda a lógica de programação do jogo (banco de dados com 20 perguntas, sistema de pontuação, verificação de acertos e barra de progresso em tempo real); - `README.md`: documentação completa do projeto (este arquivo).

4 - Tecnologias utilizadas

- HTML5: Estruturação semântica das seções e acessibilidade.
 - CSS: Estilização visual, degradês e animações de transição de telas.
 - JavaScript: Manipulação dinâmica do DOM, funções assíncronas (`setTimeout`) e gerenciamento de estados do jogo. 
- GitHub & GitHub Pages: Versionamento do código e hospedagem pública do site.

5 - Funcionalidades

- Sistema Dinâmico de Quiz: Carrega perguntas e gera botões de alternativas automaticamente via código, sem travar o navegador.
 - Barra de Progresso Visual: Atualiza a porcentagem de conclusão do desafio a cada pergunta respondida.
 - Feedback Instantâneo: Mostra na hora se o jogador tomou uma decisão sustentável correta (verde) ou incorreta (vermelho), desabilitando cliques duplos por seguransa
 - Painel de Desempenho Final: Calcula a pontuação e exibe uma mensagem motivacional personalizada de acordo com o nível de acertos do usuário.
 - Layout Limpo e Intuitivo: Focado na experiência do usuário (UX), simulando a interface de aplicativos modernos de educação.
